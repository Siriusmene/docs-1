### FFI Requirement
The new PHP Core SDK requires the FFI extension to be enabled:
- Set `ffi.enable=true` in your php.ini file
- This is required for the SDK to interface with the Rust core

### Cron Job Changes
- Legacy: `sync.php` and `send.php` scripts
- New: `StatsigSyncConfig.php` and `StatsigFlushEvents.php` scripts
