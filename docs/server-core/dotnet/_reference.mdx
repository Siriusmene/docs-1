### StatsigOptions

Configuration options for the Statsig SDK.

#### Builder Pattern
```csharp
var options = new StatsigOptions.Builder()
    .SetSpecsSyncIntervalMs(30000)
    .SetEventLoggingFlushIntervalMs(5000)
    .SetOutputLoggerLevel(OutputLogger.LogLevel.Info)
    .Build();
```
