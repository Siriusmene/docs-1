## Setup

### Basic React Router Integration

In your main application file, set up React Router with Statsig by wrapping your router with `StatsigProvider`. Replace the `sdkKey` with your Client API key from the ["API Keys" tab on the Statsig console](https://console.statsig.com/api_keys).

```tsx
import React from 'react';
import ReactDOM from 'react-dom/client';
import { createBrowserRouter, RouterProvider } from 'react-router-dom';
import { StatsigProvider } from '@statsig/react-bindings';

import HomePage from './HomePage';
import AboutPage from './AboutPage';

const router = createBrowserRouter([
  {
    path: '/',
    element: <HomePage />,
  },
  {
    path: '/about',
    element: <AboutPage />,
  },
]);

function App() {
  return (
    <StatsigProvider 
      sdkKey="client-KEY" 
      user={{ userID: "1234", email: "example@statsig.com" }}>
      <RouterProvider router={router} />
    </StatsigProvider>
  );
}

const root = ReactDOM.createRoot(document.getElementById('root'));
root.render(<App />);
```

### Alternative: Wrap Individual Routes

You can also wrap the `StatsigProvider` around your entire app and use React Router normally:

```tsx
import React from 'react';
import ReactDOM from 'react-dom/client';
import { createBrowserRouter, RouterProvider } from 'react-router-dom';
import { StatsigProvider } from '@statsig/react-bindings';

const router = createBrowserRouter([
  {
    path: '/',
    element: <HomePage />,
  },
  {
    path: '/about',
    element: <AboutPage />,
  },
]);

function App() {
  return (
    <StatsigProvider 
      sdkKey="client-KEY" 
      user={{ userID: "1234" }}>
      <div>
        <RouterProvider router={router} />
      </div>
    </StatsigProvider>
  );
}

const root = ReactDOM.createRoot(document.getElementById('root'));
root.render(<App />);
```

### Adding User Properties

The `user` prop contains properties for targeting in feature gates and experiments. See the [User (StatsigUser)](/concepts/user) doc for more information.

```tsx
const user = {
  userID: "1234",
  email: "example@statsig.com",
  customIDs: { organizationID: "org-123" },
  custom: { plan: "premium", region: "us-west" }
};

<StatsigProvider sdkKey="client-KEY" user={user}>
  <RouterProvider router={router} />
</StatsigProvider>
```
